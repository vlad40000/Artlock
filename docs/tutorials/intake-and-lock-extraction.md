# Intake And Lock Extraction

## Goal

Use the intake screen to create a project, upload or select a reference, and extract the first lock set.

## Steps

1. Open the homepage intake screen.
2. Enter a project name in `Project Identifier`.
3. Upload a new reference image or select an existing one from `Reference Assets`.
4. Confirm the selected image appears in the center preview canvas.
5. Click `Lock Extraction`.
6. Wait for the session to move through project creation, session creation, and lock extraction.
7. After extraction completes, choose the next workspace mode.

## Expected Result

- A design session is created.
- The selected reference becomes the active session reference.
- Initial locks are extracted and stored for downstream edits.

## Notes

- The `Locks` icon on intake can also trigger extraction.
- The `Images` icon opens the upload picker directly.
- If a session has already been extracted, the lock shortcut can route back into the workspace.
