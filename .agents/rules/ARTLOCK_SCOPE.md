# ARTLOCK_SCOPE.md

## Primary loop

```text
Reference -> Lock -> Delta
```

## Allowed primary work
- Add/select reference
- Extract/view/override locks
- Run bounded surgical delta
- Paint/select mask or target region
- Approve/relock/export
- References drawer
- Layers drawer
- Locks drawer
- Single-screen Studio shell

## Allowed secondary work
Secondary tools may exist behind menus/drawers:

- Trace
- Stencil
- Mockup
- Variants
- Export

## Blocked unless explicitly approved
- CRM
- billing
- scheduler
- consent forms
- aftercare
- admin panel
- marketing kit as primary flow

## Scope test
Before adding any feature, answer:

```text
Does this directly support Reference, Lock, Delta, or production output from a locked design?
```

If no, stop and ask for approval.
