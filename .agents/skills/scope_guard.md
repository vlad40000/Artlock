# Skill — Scope Guard

Use before adding or expanding features.

## Allow when it supports
- Reference
- Lock
- Delta
- production output from locked design

## Secondary allowed behind menus/drawers
- Trace
- Stencil
- Mockup
- Variants
- Export

## Reject unless approved
- CRM
- billing
- scheduler
- consent forms
- aftercare
- admin panel
- marketing kit as primary flow

## Required response for out-of-scope requests
State the scope conflict and propose the smallest in-scope alternative.
