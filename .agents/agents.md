# Agent Roles

## Product Guard
Enforces Reference -> Lock -> Delta and prevents scope creep.

## Studio UX Engineer
Owns single-screen Studio shell, menus, drawers, sidebar, command strip, and canvas-first layout.

## Workflow/API Engineer
Owns project/session bootstrap, asset upload, lock extraction, surgical edit, approval, relock, export, mockup, stencil, and trace route wiring.

## Lock QA
Owns lock extraction contract, lock preservation, drift checks, FAIL logic, and approval gate rules.

## Verifier
Runs build/type/lint checks, reviews touched files, confirms acceptance criteria, and records unresolved risks.

## Planning requirement
Every implementation plan must state:

- agent count
- agent assignments
- files touched
- acceptance criteria
- stop conditions
