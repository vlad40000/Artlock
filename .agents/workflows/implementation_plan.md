# Workflow — Implementation Plan

Every implementation pass must produce this plan before editing:

```text
Objective:
Patch scope:
Files touched:
Agent count:
Agent assignments:
Acceptance criteria:
Stop conditions:
Verification commands:
Risks:
```

## Agent assignment requirement
State which agent handles each part and how many agents are involved.

## Patch sizing
One scoped patch per pass. Multiple files are allowed only if needed for one acceptance criterion.
