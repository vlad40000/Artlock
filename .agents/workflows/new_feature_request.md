# Workflow — New Feature Request

## Step 1 — Scope check
Run `.agents/skills/scope_guard.md`.

## Step 2 — Classify
Classify as:

```text
Primary loop
Secondary hidden tool
Out of scope
Infrastructure
Bug fix
```

## Step 3 — Plan
If in scope, write an implementation plan with:

- files touched
- agent count
- agent assignments
- acceptance criteria
- stop conditions
- verification

## Step 4 — Preserve product loop
Do not allow the feature to obscure Reference -> Lock -> Delta.
